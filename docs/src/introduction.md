# Introduction to Emier Blockchain

Emier is a next-generation Layer-1 blockchain featuring modular consensus and global sharding.

## Key Features

- **Modular Consensus**: Support for multiple consensus protocols
- **Global Sharding**: 64 shards with optimized routing  
- **Quantum Security**: Post-quantum cryptography ready
- **Cross-Chain**: Universal interoperability

## Quick Start

```bash
# Build from source
cargo build --release

# Run node
./target/release/emier-node

# Use CLI
./target/release/emier-cli --version
