# Enable GitHub Pages - Manual Steps

## Quick Setup:
1. Visit: https://github.com/seekoneid-emier/emier-blockchain/settings/pages
2. Under "Source", select "Deploy from a branch"
3. Select branch: "gh-pages"
4. Folder: "/ (root)"
5. Click "Save"
